
# 堆芯热工水力单通道模型计算程序 (AP1000)

本项目是一个基于 Python 编写的压水堆（以 AP1000 为参考）堆芯热工水力学单通道模型校核程序。程序采用**等压降原理**，通过迭代计算热通道的实际流量，进而精确评估反应堆在稳态满功率工况下的核心安全限值。

## 📦 依赖项 (Dependencies)

在运行本程序之前，请确保您的 Python 环境中安装了以下依赖库：

```bash
pip install numpy matplotlib iapws
```
*   `numpy`: 用于数组计算和数值处理。
*   `matplotlib`: 用于绘制温度场、焓场、功率和 DNBR 的轴向分布图。
*   `iapws`: 国际水和蒸汽性质协会标准库，用于精确计算不同温度和压力下水/蒸汽的物性（如焓值、密度等）。

## 🚀 核心功能与输出 (Features & Outputs)

程序运行后，将在终端输出详细的计算报告，并生成包含 4 个子图的综合图表。主要输出指标严格包含以下 6 点：

1.  **堆芯冷却剂出口温度**
2.  **燃料棒表面热流密度与线功率**（包含平均值与最大值）
3.  **轴向分布数据**（热管冷却剂焓值、包壳内外表面温度、芯块中心温度随轴向高度的分布）
4.  **最高温度限值校核**（包壳表面最高温度、芯块中心最高温度 $t_{0,max}$ 校核）
5.  **DNBR 轴向变化与 MDNBR**（最小偏离泡核沸腾比，采用 W-3 经验关联式计算）
6.  **堆芯总压降**（包含摩擦、重力及各局部压降）

## 💻 使用说明 (Usage)

1.  将主程序代码保存为 `main.py`（或其他您喜欢的名称）。
2.  在终端或命令行中运行该脚本：
    ```bash
    python main.py
    ```
3.  程序将首先在控制台打印出收敛信息和 6 项核心结果数据。
4.  随后会弹出一个 Matplotlib 窗口，展示 4 张物理量轴向分布曲线图，供进一步的工程分析与报告撰写使用。

## 📊 结果判定标准参考

*   **MDNBR (最小偏离泡核沸腾比)**: 必须 $> 1.30$（典型压水堆安全限值）。
*   **芯块中心最高温度 ($t_{0,max}$)**: 必须 $< 2200 \text{ ℃}$（防止 UO2 燃料熔化）。
*   **包壳表面温度**: 允许发生局部过冷沸腾，由 Jens-Lottes 关联式限制壁温。
